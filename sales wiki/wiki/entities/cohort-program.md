## Overview

The 100xEngineers Applied AI Cohort is a 6-month live program that teaches working professionals how to build real AI systems — from generative media to production LLM apps to autonomous agents. It is not academic. Not pre-recorded. Co-founders teach live. One cohort at a time. Maximum 200-250 students.

The program deliberately targets people who already have domain expertise (engineers, founders, designers, doctors, marketers, lawyers, etc.) and helps them combine that expertise with AI. That combination is the core value proposition.

> "People who've worked 4-5 years know what real problems look like. When AI knowledge comes along with that domain experience — that combination is where real impact happens." — Zahid

|------|-------------|

## Curriculum — 4 Modules

### Module 1: Diffusion & Generative AI
*Who teaches: Sridev (CEO) + Pranay (former student, now runs AI agency)*

**What it covers:** Image and video generation end-to-end. From the math of how diffusion works, through ComfyUI workflows, LoRA training, ControlNet, IP-Adapters, to video generation (Wan 2.1, Hunyuan) and deploying custom models on Replicate.

**14 lectures — key ones:**
- How Diffusion Works (the physics of denoising, not a black box)
- ComfyUI Advanced Workflows (node-based image pipeline builder)
- Flux LoRA Training (fine-tune models on your own style/characters/product)
- Video Models (Wan 2.1, Hunyuan — AI-native video generation)
- Deploying Models on Replicate (from local to API in production)

**What students build:** Custom image generation pipelines, brand-consistent AI visuals, AI video workflows, deployed model APIs.

**Who this module hits hardest:**
- Designers, animators, filmmakers, video editors — this IS their work, supercharged
- Marketers and ad agencies — ad creative, UGC, product visuals at scale
- Founders — product mockups, pitch decks, marketing content without hiring a creative team
- Game designers — asset generation, concept art, texture workflows
- Architects, interior designers — visualization and renders

**Domain-specific use case examples:**
- Media agency: Automated Instagram B-rolls, presenter, scripts → 80% automated
- Music production: AI music videos at $25-30k/month (vs $100k before)
- E-commerce brand: Product photography and lifestyle imagery without studios

---

### Module 2: Full Stack LLM Development
*Who teaches: Siddhant (CTO, 2 prior exits)*

**What it covers:** Building real, deployable LLM-powered apps. UI (Gradio), APIs (FastAPI), databases (Supabase), prompt engineering, tool/function calling, MCP, RAG at three levels (naive → advanced → memory), and full GenAI application architecture.

**17 lectures — key ones:**
- Building APIs with FastAPI (the standard Python backend for AI apps)
- MVP Building Workshop (build a working product in one session)
- Prompt Engineering & LLM Wrappers (reliable LLM outputs in production)
- Function Calling & MCP (how AI systems talk to external tools and APIs)
- RAG — 3 levels (theory + practical builds with real data)
- Memory RAG (persistent memory across conversations)
- Building and Architecting GenAI Applications

**What students build:** LLM-powered tools connected to real data, RAG systems over company documents, custom chatbots with memory, full-stack AI apps with UI + API + database.

**Who this module hits hardest:**
- Business analysts, ops professionals — automate document processing, reporting, data extraction
- Lawyers, consultants — RAG over case files, contracts, policies
- HR professionals — automate screening, onboarding documents, policy Q&A
- Product managers — build internal tools without engineering dependency
- Marketers — content automation, competitor research pipelines
- Doctors, healthcare — RAG over medical literature, patient intake automation
- Teachers, educators — personalized learning tools, course material automation

**The compound effect framing:** 10% productivity gain per month → 60% more productive in 6 months. Sustainable AI automation, not a one-time hack.

---

### Module 3: AI Agents
*Who teaches: Siddhant (CTO)*

**What it covers:** AI agents from first principles. The ReAct loop (reason → act → observe → iterate), tool calling, multi-agent systems, agentic workflow patterns, guardrails, production deployment. Both code and no-code (n8n, LangFlow) paths.

**8 lectures — key ones:**
- Introduction to AI Agents (what makes an agent vs. a chatbot)
- Building Your First AI Agent (code + no-code parallel tracks)
- Multi-Agent Systems (specialized agents that collaborate)
- Agentic Workflow Patterns, Guardrails & Evals
- Agent Deployment & Production Patterns
- Connecting Dots from Every Module (integrates all 3 modules into one picture)

**What students build:** Autonomous research agents, multi-step workflow automation, production-deployed agents with monitoring and guardrails.

**Who this module hits hardest:**
- Founders — build AI employees that run workflows without human intervention
- Operations roles — automate multi-step business processes end-to-end
- Consultants, agencies — deliver AI-powered services to clients at scale
- Engineers — production-grade agent systems with real reliability patterns
- VCs, analysts — autonomous research and due diligence pipelines
